A PhD Research Project Directory

This repository contains the front-end architecture for the bits(~) research directory. The system is designed as a lightweight, high-performance portal that bridges the gap between static research documentation and active community engagement.

By utilizing Google Sheets as a headless CMS, the project allows for real-time updates of workshop schedules and research outputs without requiring manual code deployments.

🔬 Research Framework
This project is situated within a PhD study exploring Transmedia Storytelling as a supportive mechanism for identity exploration. It specifically focuses on students with diverse learning journeys (SEND) as they transition into Higher Education.

Affiliations: Falmouth University & University of the Arts London (UAL).

Methodology: Arts-based Educational Research (ABER) & Narrative Inquiry.

Keywords: Inclusive Pedagogy, Student Belonging, Practice-based Research.
